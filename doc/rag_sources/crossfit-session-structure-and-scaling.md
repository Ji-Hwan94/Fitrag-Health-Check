# 크로스핏 세션 구성과 스케일링 원칙

## 핵심 요약

FR-004는 운동 계획에 워밍업, 기술 연습, 근력 운동, WOD, 쿨다운을 포함하도록 요구한다. FitRAG Coach의 크로스핏 세션은 이 구조를 기본값으로 사용하고, 사용자 경험 수준과 부상 이력에 따라 동작, 부하, 반복 수, 시간을 조정한다.

## 세션 기본 구조

1. 워밍업: 5~10분의 가벼운 유산소와 동적 스트레칭으로 체온을 올린다.
2. 기술 연습: 스쿼트, 힌지, 프레스, 풀, 브레이싱, 착지 같은 기본 패턴을 짧게 연습한다.
3. 근력 운동: 목표에 맞는 주요 리프트 또는 보조 운동을 낮거나 중간 볼륨으로 수행한다.
4. WOD: 사용자의 수준에 맞춰 시간, 반복 수, 부하를 조정한 메인 컨디셔닝을 수행한다.
5. 쿨다운: 호흡 정리와 주요 근육군 스트레칭으로 세션을 마무리한다.

## 스케일링 규칙

- 부하 스케일링: 바벨 중량을 낮추거나 덤벨, 케틀벨, 맨몸 동작으로 바꾼다.
- 반복 수 스케일링: 목표 시간 안에 자세가 무너지지 않도록 반복 수를 줄인다.
- 동작 스케일링: 풀업은 링로우, 푸시업은 인클라인 푸시업, 박스 점프는 스텝업으로 대체한다.
- 시간 스케일링: 초보자는 AMRAP 또는 인터벌 시간을 짧게 설정한다.
- 충격 스케일링: 점프와 달리기는 자전거, 로잉, 스키에르그, 걷기로 바꿀 수 있다.

## 초보자 우선 원칙

초보자에게는 기초 동작 중심의 낮은 강도 계획을 우선 제공한다. 사용자가 WOD를 완료하는 것보다 통증 없이 안정적인 자세를 유지하는 것이 더 중요하다.

초보자 WOD 예시는 8~12분 AMRAP, EMOM, 또는 짧은 라운드 형식이 적합하다. 복잡한 역도 동작, 고반복 체조 동작, 고중량 리프트는 충분한 기술 연습 없이 추천하지 않는다.

## 운동 계획 생성 시 체크리스트

- 하루 세션 안에 워밍업, 기술, 근력, WOD, 쿨다운이 모두 있는가?
- 주간 계획 안에 회복일 또는 낮은 강도 대체 운동이 있는가?
- 목표와 무관한 고난도 동작이 포함되지 않았는가?
- 부상 이력에 맞는 대체 운동이 명시되었는가?
- 사용자가 이해하기 쉬운 표현으로 강도와 주의 사항을 설명했는가?

## 답변에 포함할 근거 요약 예시

크로스핏 계획은 세션 구조를 분명히 나누고, 초보자와 부상 이력이 있는 사용자에게는 동작 난도, 반복 수, 부하, 충격량을 낮추는 스케일링을 적용해야 한다.
