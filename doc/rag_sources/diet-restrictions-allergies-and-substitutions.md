# 식단 제한 조건, 알레르기, 대체 식품 가이드

## 핵심 요약

FR-005는 사용자의 알레르기, 제한 조건, 선호 음식을 식단 계획에 반영하도록 요구한다. FR-006은 추천 식단에 필요한 식재료 리스트와 검색 키워드를 생성하도록 요구한다. 따라서 RAG 문서는 제한 조건별 대체 식품과 장보기 키워드 생성 기준을 함께 제공해야 한다.

## 알레르기 우선 정책

알레르기로 입력된 식품은 추천 메뉴, 재료 리스트, 쇼핑 키워드에서 제외한다. 알레르기 가능성이 있는 재료가 소스, 가공식품, 보충제에 숨어 있을 수 있으므로 제품 구매 전 라벨 확인을 안내한다.

알레르기 정보가 모호하면 해당 식품군을 보수적으로 제외하고 대체 식품을 제안한다. 예를 들어 "유제품 불편"이 입력되면 우유, 그릭요거트, 치즈, 유청 단백질을 바로 추천하지 않는다.

## 제한 조건별 대체 식품

- 채식: 닭가슴살, 생선, 살코기 대신 두부, 템페, 콩류, 렌틸콩, 병아리콩, 식물성 단백질 보충제를 사용한다.
- 유제품 제한: 그릭요거트, 우유, 유청 단백질 대신 무가당 두유, 락토프리 제품, 식물성 단백질, 계란 또는 두부를 사용한다.
- 글루텐 제한: 일반 빵, 밀가루 면 대신 쌀밥, 현미밥, 감자, 고구마, 쌀국수, 메밀 100% 제품을 사용한다.
- 생선 제한: 연어, 참치 대신 닭가슴살, 계란, 두부, 콩류, 살코기를 사용한다.
- 견과류 알레르기: 아몬드, 땅콩, 견과류 버터를 제외하고 올리브오일, 아보카도, 씨앗류도 개별 안전 여부를 확인한 뒤 사용한다.

## 선호 음식 반영

선호 음식은 전체 영양 목표를 해치지 않는 범위에서 메뉴에 포함한다. 비선호 음식은 가능한 대체 식품으로 바꾼다. 다만 선호 음식이 고열량 디저트, 당음료, 튀김류라면 매일 식단의 중심으로 배치하지 않는다.

## 장보기 키워드 생성 기준

식재료 검색 키워드는 구체적인 식품명과 용도를 함께 포함한다. 예를 들어 "닭가슴살 식단", "무가당 그릭요거트", "현미밥 즉석", "두부 단백질 식단", "냉동 블루베리"처럼 사용자가 바로 구매 후보를 탐색할 수 있게 만든다.

알레르기 또는 제한 조건이 있는 경우 검색 키워드에도 제한 조건을 반영한다. 예를 들어 유제품 제한 사용자는 "식물성 단백질 보충제", 글루텐 제한 사용자는 "글루텐프리 오트"처럼 생성한다.

## 답변에 포함할 근거 요약 예시

식단 추천은 알레르기와 제한 조건을 먼저 제외한 뒤 목표 영양소에 맞는 대체 식품을 제안해야 한다. 장보기 키워드는 식품명과 제한 조건을 함께 반영해 생성한다.
