# 요구사항 정의서

FitRAG Coach MVP 개발을 위한 기능, 비기능, 정책 요구사항을 정의한다.

## 1. 서비스 목적

사용자가 신체 정보와 목표를 입력하면 크로스핏 중심의 주간 운동 계획, 목표 기반 식단 계획, 식재료 쇼핑 링크, RAG 기반 추천 근거를 제공한다.

## 2. 사용자 유형

### 일반 사용자

- 신체 정보와 목표를 입력한다.
- 추천 운동 계획과 식단 계획을 확인한다.
- 식재료 리스트와 쇼핑 검색 링크를 확인한다.
- 추천 근거와 주의 사항을 확인한다.

### 관리자 또는 운영자

- RAG 문서 소스를 등록하고 관리한다.
- 추천 품질과 사용자 피드백을 확인한다.
- 위험 조건, 금지 표현, 면책 문구 정책을 관리한다.

## 3. 기능 요구사항

### FR-001 회원 및 사용자 프로필

- 사용자는 이메일 기반 계정을 생성할 수 있어야 한다.
- 사용자는 성별, 나이, 키, 체중, 운동 경험 수준을 입력할 수 있어야 한다.
- 사용자는 부상 이력, 알레르기, 식단 제한 조건을 선택 입력할 수 있어야 한다.

### FR-002 목표 입력

- 사용자는 감량, 증량, 근육 증가, 체지방 감소, 체력 향상 중 목표를 선택할 수 있어야 한다.
- 사용자는 목표 체중, 목표 근육량, 목표 체지방량, 목표 기간을 입력할 수 있어야 한다.

### FR-003 신체 상태 분석

- 시스템은 BMI를 계산해야 한다.
- 시스템은 목표까지 필요한 체중 변화량을 계산해야 한다.
- 시스템은 목표 기간과 건강한 변화 속도를 비교해야 한다.
- 위험 조건이 있는 경우 전문가 상담 안내를 표시해야 한다.

### FR-004 운동 계획 추천

- 시스템은 운동 경험 수준과 주당 가능 횟수를 기준으로 주간 운동 계획을 생성해야 한다.
- 운동 계획은 워밍업, 기술 연습, 근력 운동, WOD, 쿨다운을 포함해야 한다.
- 초보자에게는 낮은 강도와 기초 동작 중심 계획을 우선 제공해야 한다.
- 계획에는 회복일 또는 대체 운동이 포함되어야 한다.

### FR-005 식단 계획 추천

- 시스템은 일일 권장 칼로리와 단백질, 탄수화물, 지방 목표량을 계산해야 한다.
- 시스템은 아침, 점심, 저녁, 간식 예시를 제공해야 한다.
- 사용자의 알레르기, 제한 조건, 선호 음식을 반영해야 한다.
- 극단적인 칼로리 제한 식단은 제공하지 않아야 한다.

### FR-006 장보기 연동

- 시스템은 추천 식단에 필요한 식재료 리스트를 생성해야 한다.
- 시스템은 식재료별 검색 키워드를 생성해야 한다.
- 시스템은 쿠팡 검색 URL 또는 일반 쇼핑 검색 URL을 제공해야 한다.

### FR-007 RAG 근거 제공

- 시스템은 운동 및 영양 추천 시 관련 문서를 검색해야 한다.
- 추천 결과에는 근거 요약과 출처 정보가 포함되어야 한다.
- 검색 근거가 부족하면 확정적인 표현을 피해야 한다.

### FR-008 추천 결과 저장

- 사용자는 생성된 운동 계획과 식단 계획을 저장할 수 있어야 한다.
- 사용자는 최근 추천 결과를 다시 조회할 수 있어야 한다.

## 4. 비기능 요구사항

### 성능

- 첫 추천 결과는 일반적인 입력 기준 15초 이내 반환을 목표로 한다.
- 저장된 추천 결과 조회는 2초 이내 반환을 목표로 한다.

### 신뢰성

- LLM 응답 실패 시 재시도 또는 기본 안내 문구를 제공해야 한다.
- RAG 검색 실패 시 근거 부족 상태를 명확히 표시해야 한다.

### 보안 및 개인정보

- 건강 정보는 인증된 사용자만 접근할 수 있어야 한다.
- 민감한 건강 데이터는 DB 저장 시 최소화한다.
- API 요청에는 인증 토큰을 사용한다.

### 사용성

- 초보자도 이해할 수 있는 표현을 사용한다.
- 운동 강도, 주의 사항, 대체 운동을 명확히 표시한다.

## 5. 정책 요구사항

- 의료 진단, 치료, 질병 판정 표현을 사용하지 않는다.
- 질환, 임신, 섭식장애, 심각한 부상 이력이 있는 경우 전문가 상담을 권장한다.
- 무리한 감량, 과훈련, 극단적 식단을 추천하지 않는다.
- 추천 내용은 참고용이며 최종 실행은 사용자 상태에 맞게 조정하도록 안내한다.

## 6. MVP 범위

### 포함

- 사용자 프로필 입력
- 목표 입력
- BMI 및 기본 영양 계산
- 주간 크로스핏 운동 계획 생성
- 하루 식단 예시 생성
- 식재료 리스트와 검색 URL 생성
- RAG 기반 근거 검색 및 표시

### 제외

- 결제 기능
- 실제 쇼핑몰 장바구니 자동 추가
- 웨어러블 연동
- 트레이너 관리자 페이지
- 의료 상담 기능
- 운동 기록 자동 분석
